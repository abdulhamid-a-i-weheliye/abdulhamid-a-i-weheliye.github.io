Here is a selection of projects I've worked on

JavaScript Games Website

This was a smaller project I worked on where I built a webpage that allowed a user to select from 1 of 3 options. The first was a simple guess who game that used arrays and functions to randomly select a name. The user was then given 2 chances to guess from a list of 4 randomly chosen names, including the selected answer. The other two options were a palindrome check, which took a word and would return if the word was a palindrome or not. The last was a calculator function which took two numbers and allowed the user to select from four calculation types.

Mystery Falls CYOA Game

Mystery Falls is a Choose Your Own Adventure (CYOA) game I helped lead a team of 4 to build. I primarily worked on designing the main dialogue and shoice selection systems utilizing a combination of HTML, CSS, and JavaScript.
